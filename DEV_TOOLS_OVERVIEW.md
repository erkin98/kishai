## Quick Reference

### First Time Setup
```bash
# 1. Install tools
curl -LsSf https://astral.sh/uv/install.sh | sh
cargo install just

# 2. Setup project
./SETUP.sh

# 3. Start coding!
source .venv/bin/activate
just dev
```

### Most Used Commands
```bash
just dev          # Start server (auto-reload)
just test         # Run tests
just fmt          # Format code
just check        # All quality checks
just clean        # Clean up
```

### Before Committing
```bash
just fmt          # Format code
just check        # Run all checks
just test         # Run tests
git add .
git commit -m "feat: your changes"
```

### Troubleshooting
```bash
just verify       # Check setup
just info         # Show environment info
just clean-all    # Nuclear option - clean everything
just setup        # Recreate environment
```
